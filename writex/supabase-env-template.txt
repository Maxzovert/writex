# How to get these values:
# 1. Go to your Supabase Console (https://supabase.com/dashboard)
# 2. Select your project
# 3. Go to "Settings" → "API"
# 4. Copy the Project URL and anon (public) key
# 5. Make sure to remove any quotes from the values!

# Common mistakes to avoid:
# ❌ VITE_SUPABASE_URL="https://my-project.supabase.co"  (with quotes)
# ✅ VITE_SUPABASE_URL=https://my-project.supabase.co    (without quotes)
